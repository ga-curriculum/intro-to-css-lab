## Setup

Open your Terminal application and navigate to your `~/code/ga/labs` directory:

```bash
cd ~/code/ga/labs
```

<!-- tktk if starting from scratch  -->

Make a new repository on [GitHub](https://github.com/) named press-release-lab.

<!-- tktk -->

Clone a copy of your remote repo locally by using the `git clone` command:

```bash
git clone https://github.com/<your-username>/press-release-lab.git
```

> 📚 Note: In the link above, where it says `<your-username>`, you should see the username from your GitHub account.

Next, `cd` into your new cloned directory, `press-release-lab`:

```bash
cd press-release-lab
```

Then, create an `index.html` file and a `style.css` file. Both of these files will be used for this lab: 

```bash
touch index.html css/style.css
```

Open the project's folder in your code editor:

```bash
code .
```
